# Apache-Logs-Viewer 🔎
A Python application for parsing/managing Apache logs

## ✅ Program modes:
``` HTML
-h, -b, -i, -a
```

## 💬 List of available functions:
- Logs Viewer
- Sort by IP/Date
- Group by IP+Status
- Group by Date or Period
- Clear logs from DB
- Load logs to DB

## 🔓 Installation:
- Open setup.bat to import libraries.
- Open settings.py
- Edit "Database name", "Active folder", "File mask". (if needed)

## 📝 Example: 
<p>You can open it in different states:</p>
Help:
```python main.py -h```
Batch:
```python main.py -b```
Interactive:
```python main.py -i```
API:
```python main.py -a (**args)```
